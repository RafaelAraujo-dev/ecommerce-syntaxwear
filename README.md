# SyntaxWear - Tênis e Sneakers Online

![Banner Hero](images/banners/hero.jpg)

## Descrição do Projeto

SyntaxWear é uma landing page de e-commerce focada na venda de tênis e sneakers online, oferecendo conforto e estilo em cada passo. O projeto apresenta um design moderno e responsivo, ideal para dispositivos móveis e desktops.

## Funcionalidades

*   **Página Principal:** Exibição de produtos em destaque, categorias e informações essenciais.
*   **Navegação:** Menu de navegação claro com categorias (Masculino, Feminino, Outlet) e acesso rápido (Minha conta, Ajuda, Carrinho, Nossas lojas, Sobre).
*   **Seção Hero:** Banner principal com chamada para ação, destacando um produto ou promoção.
*   **Categorias de Produtos:** Cards interativos para explorar diferentes categorias como Casual, Esporte, Moderno e Futurista.
*   **Grid de Produtos:** Layout em grid para exibir produtos em destaque.
*   **Rodapé Completo:** Informações de contato, links úteis, redes sociais e formulário de newsletter.
*   **Responsividade:** Design adaptável para diferentes tamanhos de tela (desktop e mobile).

## Tecnologias Utilizadas

*   **HTML5:** Para a estruturação do conteúdo da página web.
*   **CSS3:** Para a estilização e responsividade do site, utilizando uma abordagem modular com arquivos de reset, variáveis, base e componentes.

## Como Visualizar o Projeto

Para visualizar este projeto, basta abrir o arquivo `index.html` em seu navegador web. Não é necessário nenhum servidor ou configuração adicional.

1.  Clone o repositório (se aplicável) ou baixe os arquivos do projeto.
2.  Navegue até a pasta raiz do projeto.
3.  Abra o arquivo `index.html` com seu navegador preferido (Chrome, Firefox, Edge, etc.).
